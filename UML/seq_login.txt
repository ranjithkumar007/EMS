title "UseCase - Login"

actor User
activate User
participant SoftwareGUI
activate SoftwareGUI
database UserRecord
User -> SoftwareGUI : Start Application
SoftwareGUI --> User : Register/Log-In
User -> SoftwareGUI : Request for Login
SoftwareGUI --> User : Enter Username
User -> SoftwareGUI : Username
SoftwareGUI --> User : Enter Password
User -> SoftwareGUI : password
SoftwareGUI --> UserRecord : Authentication Check
deactivate SoftwareGUI
activate UserRecord
UserRecord -> SoftwareGUI : Response
deactivate UserRecord
activate SoftwareGUI
SoftwareGUI --> User : Successful Login
note left : Login Complete
deactivate User
deactivate SoftwareGUI
deactivate UserRecord
